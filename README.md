## 节假日API

| 名称                 | 地址                           |
| -------------------- | ------------------------------ |
| 大小周-指定日期      | api/holiday/info/single/{date} |
| 大小周-当日          | api/holiday/info/single        |
| 大小周-当日-节后单休 | api/holiday/info/singlee       |
| 大小周-次日          | api/holiday/info/single/next   |
| 大小周-次日-节后单休 | api/holiday/info/singlee/next  |
| 双休-指定日期        | api/holiday/info/{date}        |
| 双休-当日            | api/holiday/info               |
| 双休-次日            | api/holiday/info/next          |

date用yyyy-MM-dd格式
